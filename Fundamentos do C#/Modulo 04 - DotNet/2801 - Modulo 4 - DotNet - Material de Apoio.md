## dotnet CLI

**Criando um novo projeto**
_dotnet cli_

- Novo Console Application

```
    dotnet new console
```

- Nova Class Library

```
    dotnet new classlib
```

- Novo projeto ASP.NET Core

```
    dotnet new web
```

- Novo projeto ASP.NET Core

```
    dotnet new mvc
```

- Novo projeto ASP.NET Core

```
    dotnet new webapi
```

- Novo projeto Microsoft Test

```
    dotnet new mstest
```
